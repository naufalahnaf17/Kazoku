Prototype Kazoku

## Cara Run Project

Clone Projectnya Terlebih Dahulu

```bash
git clone https://github.com/naufalahnaf17/Kazoku.git
```

Install Dependencies

```bash
npm install
npx expo prebuild
```

IOS Required (Android Skip)

```bash
cd ios
pod install
```

Run Project (Android)

```bash
npx expo run:android
```

Run Project (IOS)

```bash
npx expo run:ios
```
